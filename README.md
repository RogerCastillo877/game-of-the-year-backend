# Goty Firebase

## Config Firebase
npm install -g firebase-tools
* firebase login
* firebase init
* firebase serve

** firebase init emulators
** firebase emulators:start --only functions

## To make changes
cd functions
npm run build
* tsc --watch

## Install express
cd functions
npm install express cors
npm install @types/express --save-dev
npm install @types/cors --save-dev
